Projeto de Desenvolvimento Front-End

Este repositório foi criado para armazenar e organizar todas as atividades práticas e exercícios desenvolvidos ao longo da disciplina de Desenvolvimento Front-End na instituição UniSenai-PR, como parte do curso de Engenharia de Software.

Objetivo do Projeto
O principal objetivo deste projeto é centralizar o histórico de aprendizado das tecnologias essenciais da Web: HTML5, CSS3 e introdução ao JavaScript.

Organização do Conteúdo
Para demonstrar a evolução real e cronológica do aprendizado, a estrutura do projeto foi dividida da seguinte forma:

* Paginas de Apresentação e Menus: As telas de índice, navegação e os menus dos capítulos foram totalmente customizados pelo aluno utilizando técnicas modernas de CSS e layouts responsivos.
  
* Exercícios de Sala de Aula: Todas as páginas construídas de forma prática durante as aulas foram mantidas estritamente em seu estado original. Essa escolha foi feita de forma consciente para preservar o registro fiel do progresso técnico e o domínio gradativo de cada conceito no momento em que ele foi ensinado.

Tecnologias Utilizadas

* HTML5: Estruturação semântica do conteúdo.
* CSS3: Estilização, criação de temas e técnicas de layout.
* JavaScript: Lógica inicial para manipulação do DOM e dinamismo das páginas.
